# Screenshots Folder

## 📸 How to Add Your Screenshots

1. Take screenshots of your deployed application at: https://medical-chatbot-179320.vercel.app
2. Save them with the following names:
   - `screenshot1.png` - Chat interface (login or main chat screen)
   - `screenshot2.png` - Medical Q&A in action (conversation example)

3. Place both images in this `screenshots/` folder

## 📏 Recommended Image Specifications

- **Format:** PNG or JPG
- **Resolution:** 1920x1080 or similar (16:9 ratio)
- **File Size:** < 5MB each
- **Content Suggestions:**
  - Screenshot 1: Login page or empty chat interface
  - Screenshot 2: Active conversation showing medical question and AI response

## 🎨 Tips for Great Screenshots

- Use a clean browser window (hide bookmarks bar)
- Show the full application interface
- Include interesting medical questions/answers
- Ensure text is readable
- Use light mode for better clarity (if available)

---

Once you add your images, they will automatically appear in the main README.md file!
